commands for run the Automation :=

1: Check node is installed or not : node -v

2: check python is installed or not : python --version

3: install package.json : npm install

4: install playwright : npx playwright install

5: install camelot : pip install camelot-py[cv] ghostscript

6: run the app : npm start

7: make .exe : npm run package

How the app works (3-step flow):
- Step 1: Click Start → Login + Find Claim + Fill Hospital/Lab/Chemist sections
- Step 2: Click Fill Remaining Form → Fill Home Visit + Others + Office/School/College
- Step 3: Click Run Conclusion → Fill Verification Conclusion + upload documents
