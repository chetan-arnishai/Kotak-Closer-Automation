How to Run / टूल कसे चालवायचे

  1. Enter Claim Number — no spaces allowed.
    क्लेम नंबर टाका — स्पेस नसाव्यात.
  2. Click Browse PDF to select the Main Report only.
    फक्त Main Report PDF निवडा.
  3. Place all other PDFs in the upload_documents folder on your Desktop.
    They will be loaded automatically — no need to select them manually.
    इतर सर्व PDF Desktop वरील "upload_documents" फोल्डरमध्ये ठेवा — ते आपोआप लोड होतात.
  4. Click Start — Login will happen automatically.
    Enter OTP in the browser when prompted.
    Start दाबा — Login आपोआप होतो. OTP टाका.
  5. After OTP is validated, navigate to the form page manually in the browser.
    Form fill will begin automatically.
    OTP नंतर ब्राउझरमध्ये फॉर्म पेजवर जा — फॉर्म भरणे आपोआप सुरू होईल.
  6. Submit the form manually. Then click Run Conclusion in the app.
    फॉर्म मॅन्युअली सबमिट करा. मग Run Conclusion दाबा.
  7. Do NOT touch keyboard or mouse during automation.
    ऑटोमेशन सुरू असताना हात लावू नका.
